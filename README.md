<h1 align="center">DB First</h1>

Modellazione di una tabella database per la gestione delle auto usate messe in vendita da un concessionario.

---

## Obiettivo

Progettare la struttura di una tabella relazionale in grado di memorizzare tutte le informazioni principali relative alle auto usate, definendo correttamente campi, tipi di dato e vincoli.

---

## Struttura della tabella

La tabella è progettata per rappresentare le auto usate disponibili presso un concessionario, includendo dati anagrafici, tecnici e commerciali.

Il dettaglio completo della struttura è disponibile nel file:
- auto_usate.txt



## Concetti applicati

- Modellazione **DB-first** di base
- Scelta dei tipi di dato appropriati
- Definizione di chiave primaria
- Applicazione di vincoli minimi (unicità, obbligatorietà)
- Utilizzo di valori di default
- Gestione di campi opzionali


---

## Tecnologie / Strumenti

- Database relazionale
- SQL (progettazione)
